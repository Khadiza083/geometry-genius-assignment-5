Question-1: What is DOM ? and what is the purpose of DOM?

Answer:

The full meaning of DOM is 'Document Object Model'. the purpose of DOM is to provide a strurcture by using HTML elements as object

Question-2: How will you select HTML elements using DOM? Name the DOM methods.

Answer:

By using DOM selector methods I can select HTML elements.
Name of the DOM methods:
  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()

Question-3: What is event bubble?

Answer:

Bubble event happens when an element receives an event, that events bubbles up to its parent. It provides all HTML elements from trigger element to its parent.

Question-4: What is a callback function and why will you use it?

Answer:

A callback function is a function that is passed as an argument to another function. When I need one function to only after another function this time i will use it.
